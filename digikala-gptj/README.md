# سامانه تحلیل صورت‌حساب دیجی‌کالا با GPT‑J

این ریپو یک سایت حرفه‌ای (GitHub Pages) برای آپلود فایل اکسل صورت‌حساب دیجی‌کالا و اجرای تحلیل مالی با کمک GPT‑J فراهم می‌کند. خروجی به‌صورت فایل اکسل سه‌شیتی منتشر می‌شود:

- Summary: جمع کل‌ها، سود کل، حاشیه سود کل، تعداد فروش نهایی
- Itemized: ردیف به ردیف با DKPC، مبالغ، سود هر کالا، حاشیه سود — مرتب‌شده بر اساس DKPC
- GPTJ_Notes: یادداشت‌های GPT‑J برای هر ردیف

## راه‌اندازی سریع
1. Secrets → HF_API_TOKEN را اضافه کنید.
2. Settings → Pages → Branch: gh-pages
3. فایل اکسل را در inputs/latest.xlsx آپلود کنید.
4. خروجی در outputs/latest.xlsx و صفحه‌ی سایت قابل دانلود خواهد بود.
